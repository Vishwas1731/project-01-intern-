# 📊 HR Analytics – Employee Attrition Prediction

## 🚀 Project Overview

This project focuses on predicting employee attrition using **Machine Learning and Data Analytics**. The goal is to help organizations identify employees at risk of leaving and take preventive actions.

---

## 📌 Objectives

* Analyze employee data to find attrition patterns
* Build predictive models for employee turnover
* Provide actionable business insights

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Seaborn, Matplotlib)
* Scikit-Learn (Machine Learning)
* SHAP (Explainable AI)
* Power BI (Dashboard & Visualization)

---

## 📂 Dataset

* IBM HR Analytics Employee Attrition Dataset
* 1470 records and 30+ features

---

## ⚙️ Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building
5. Model Evaluation
6. SHAP Explainability
7. Attrition Risk Prediction
8. Power BI Dashboard
9. Streamlit Web App

---

## 🤖 Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Gradient Boosting

🏆 Best Model: Gradient Boosting
📈 Accuracy: ~89%

---

## 📊 Power BI Dashboard Features

* Attrition Rate KPI
* Attrition by Department
* Salary vs Attrition Analysis
* Employee Risk Segmentation
* Attrition Trend Analysis

---

## 🔍 Key Insights

* Overtime significantly increases attrition
* Employees with low salaries are more likely to leave
* Early-career employees have higher attrition rates
* Job satisfaction plays a major role

---

## 💡 Business Recommendations

* Reduce employee workload
* Improve compensation packages
* Focus on employee engagement
* Provide career development opportunities
